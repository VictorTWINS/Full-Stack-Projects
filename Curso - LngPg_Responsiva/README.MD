# 🚀 Minha Primeira Landing Page Responsiva

Bem-vindo ao repositório que marca o início oficial da minha jornada no desenvolvimento de software. Este projeto consiste em uma landing page construída do zero, focada em fundamentos sólidos, semântica e adaptabilidade.

## 📌 Sobre o Projeto
O objetivo principal foi aplicar os conhecimentos iniciais de Front-End para criar uma interface funcional, moderna e totalmente responsiva. Busquei ir além do visual, focando na organização do código e na aplicação de padrões utilizados por profissionais da área.

## 🛠️ Tecnologias Utilizadas
*   **HTML5:** Estruturação semântica para melhor acessibilidade e SEO.
*   **CSS3:** Estilização avançada, utilizando variáveis (Custom Properties) e layouts modernos.
*   **Flexbox/Grid:** Para garantir que a estrutura seja flexível em qualquer tamanho de tela.
*   **Mobile First:** Desenvolvimento focado primeiro em dispositivos móveis, escalando para desktop.

## 🧠 Boas Práticas Aplicadas
Neste projeto, priorizei a qualidade do código através de:
- **Semântica:** Uso correto de tags como `<header>`, `<main>`, `<section>` e `<footer>`.
- **Sintaxe Limpa:** Código identado e organizado para facilitar a leitura.
- **Responsividade:** Uso de *Media Queries* para garantir uma experiência fluida em smartphones, tablets e monitores.
- **Naming Convention:** Organização de classes CSS seguindo padrões de fácil manutenção.

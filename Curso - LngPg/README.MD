# 🧪 Projeto Laboratório: Minha Primeira Landing Page

Este repositório registra o meu **ponto de partida** prático na programação. Antes de focar em padrões de mercado e arquitetura de código, precisei "sujar as mãos" para entender como as peças se encaixam.

## 🎯 O Objetivo
O foco aqui foi exclusivamente **aplicação e experimentação**. Eu queria tirar a teoria do papel e transformar o que aprendi sobre HTML5 e CSS3 em algo visual e funcional.

## 🛠️ O que foi testado:
- Estruturação básica de documentos HTML.
- Estilização de elementos (cores, fontes e posicionamento).
- Entendimento do fluxo de renderização no navegador.
- Sintaxe inicial das linguagens.

## ⚠️ Disclaimer (Aviso Importante)
Este é um projeto de aprendizado técnico inicial. Por ser um ambiente de testes, **não tive como prioridades**:
- **Responsividade:** O layout foi pensado para telas específicas, focando no aprendizado da propriedade antes da adaptação.
- **Naming Convention:** A nomenclatura de classes e IDs seguiu um fluxo intuitivo de estudo, sem a aplicação rigorosa de metodologias como BEM ou similares.
- **Boas Práticas de Arquitetura:** O objetivo foi o "fazer funcionar" para validar o conhecimento.

## 📈 Evolução
Este projeto foi o degrau necessário para que eu pudesse avançar para o desenvolvimento de interfaces mais robustas, agora sim focando em **Clean Code**, **Mobile First** e **Semântica Profissional**. 

Ele serve como um lembrete do meu progresso e da importância de dominar a base antes de construir sistemas complexos.

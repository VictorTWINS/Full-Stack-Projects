# 🟡 Módulo 02: JavaScript Básico

Este diretório contém exercícios e pequenos projetos focados nos fundamentos da linguagem JavaScript. É aqui que a lógica de programação ganha vida no navegador.

## 🧠 O que estou estudando neste módulo:
* **Fundamentos da Lógica**: Variáveis, tipos de dados, operadores e estruturas condicionais (if/else, switch).
* **Estruturas de Repetição**: For, While e métodos de array.
* **Funções**: Declaração, escopo e Arrow Functions.
* **Manipulação de DOM**: Como selecionar elementos, alterar estilos e textos dinamicamente.
* **Eventos**: Captura de cliques, inputs de formulários e movimentos do mouse.

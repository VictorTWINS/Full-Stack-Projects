🚀 Full-Stack Projects - Jornada de Aprendizado
Este repositório centraliza todos os meus projetos e estudos desenvolvidos durante a minha formação Full-Stack. O objetivo é documentar a evolução técnica em cada módulo, desde a base do desenvolvimento web até aplicações complexas.

📌 Estrutura do Repositório
O repositório está organizado por módulos temáticos para facilitar a navegação:

01. HTML e CSS: Focado em semântica, acessibilidade e estilização avançada (Flexbox, Grid, Responsividade).

02. JavaScript - Básico: Lógica de programação, manipulação de DOM e eventos simples.

03. JavaScript - Avançado: Programação assíncrona, APIs e conceitos modernos de ES6+.

04. Projetos Completos: Aplicações que integram Front-End e Back-End.

🛠️ Destaque do Módulo 01: Landing Page Blend Burguer
O primeiro grande projeto concluído foi uma Landing Page para uma Hamburgueria, onde apliquei conceitos fundamentais de design e desenvolvimento.

Tecnologias Utilizadas:
HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização personalizada, incluindo:

Uso da fonte Montserrat para legibilidade.

Efeitos de transition e hover para interatividade nos botões.

Design não é responsivo **(ainda)** para diferentes tamanhos de tela.

O que aprendi neste projeto:
Organização de arquivos e ativos (Assets).

Trabalho com formulários de reserva e validações simples.

Controle de versão com Git e GitHub para organizar o fluxo de trabalho.
